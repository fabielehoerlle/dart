# Variáveis

## Tipos:

```
1 - String (texto) variavelNome = "Fabiele";
2 - int (número inteiro) valorVariavel = 10; 
3 - bool (verdadeiro ou falso) ehVerdadeiro = true;
4 - List<String> listaDePalavras = ["Fabiele", "Hoerlle"];
```

## Operador:

```
1 - print(variavelNome;)
2 - print (valorVariavel);
3 - print (ehVerdadeiro);
4 - print('${listaDePalavras[0]} - ${listaDePalavras[1]}');
```
