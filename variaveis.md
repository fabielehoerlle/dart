# Variáveis

## Tipos:

* String (texto) variavelNome = "Fabiele";
* int (número inteiro) valorVariavel = 10; 
* bool (verdadeiro ou falso) ehVerdadeiro = true;
* List<String> listaDePalavras = ["Fabiele", "Hoerlle"];

## Operador:

* print(variavelNome;)
* print (valorVariavel);
* print (ehVerdadeiro);
* print('${listaDePalavras[0]} - ${listaDePalavras[1]}');
